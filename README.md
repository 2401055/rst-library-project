# مكتبة RST - RST Library

مشروع تطبيق ويب لمكتبة رقمية يتضمن واجهة أمامية حديثة وخلفية برمجية قوية.

## المميزات

✅ نظام تسجيل دخول وإنشاء حساب  
✅ إدارة الكتب والبحث المتقدم  
✅ نظام المفضلة  
✅ إدارة الفعاليات  
✅ نظام تقديم الشكاوى والملاحظات  
✅ قاعدة بيانات SQLite  
✅ واجهة مستخدم عصرية وسهلة الاستخدام  

## المتطلبات

- Python 3.7+
- pip (مدير الحزم في Python)
- متصفح ويب حديث

## التثبيت والتشغيل

### 1. تثبيت المكتبات المطلوبة

```bash
pip install flask flask-cors flask-sqlalchemy pyjwt
```

### 2. تشغيل التطبيق

```bash
python app.py
```

سيبدأ الخادم على `http://localhost:5000`

### 3. فتح التطبيق

افتح ملف `index.html` في متصفحك أو استخدم خادم محلي:

```bash
# باستخدام Python 3
python -m http.server 8000
```

ثم افتح `http://localhost:8000/index.html`

## الملفات

### index.html
- يحتوي على الواجهة الأمامية الكاملة
- HTML: هيكل الصفحة
- CSS: التنسيقات والتصاميم
- JavaScript: البرمجة التفاعلية والاتصال بـ API

### app.py
- تطبيق Flask للخلفية البرمجية
- إدارة قاعدة البيانات باستخدام SQLAlchemy
- API endpoints للتعامل مع البيانات
- نظام المصادقة باستخدام JWT

## API Endpoints

### المصادقة
- `POST /api/register` - تسجيل مستخدم جديد
- `POST /api/login` - تسجيل الدخول
- `GET /api/user/profile` - الحصول على بيانات المستخدم

### الكتب
- `GET /api/books` - الحصول على قائمة الكتب
  - معاملات اختيارية: `search`, `category`

### الفعاليات
- `GET /api/events` - الحصول على قائمة الفعاليات

### المفضلة
- `GET /api/user/favorites` - الحصول على الكتب المفضلة
- `POST /api/user/favorites` - إضافة كتاب للمفضلة

### الشكاوى
- `POST /api/complaints` - تقديم شكوى أو ملاحظة

- 4 فعاليات قادمة

## ملاحظات أمنية

⚠️ **تغيير المفتاح السري:**
قبل نشر التطبيق في الإنتاج، غيّر قيمة `SECRET_KEY` في `app.py`:

```python
app.config['SECRET_KEY'] = 'your_secret_key_change_in_production'
```

استخدم مفتاح عشوائي قوي بدلاً من القيمة الافتراضية.

## استكشاف الأخطاء

### الخطأ: "Connection refused"
تأكد من أن خادم Flask يعمل على `http://localhost:5000`

### الخطأ: "CORS error"
تأكد من أن `flask-cors` مثبت وأن `CORS(app)` موجود في `app.py`

### الخطأ: "Database locked"
أغلق جميع نسخ التطبيق وحاول مرة أخرى
